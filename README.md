## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/RiyaAhuja-182/accredian-assignment.git
cd 
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
npm start
```
